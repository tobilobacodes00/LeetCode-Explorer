The problem checks if an object has any properties or if an array has any elements. You can use Object.keys(obj).length to get the number of keys in the object. If the length is 0, then it's empty.

Works for both arrays and objects since arrays are objects in JavaScript.

var isEmpty = function(obj) {
    return Object.keys(obj).length === 0;
