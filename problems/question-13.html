We can use setTimeout inside a Promise to delay execution. This allows us to create a delay function similar to sleep() in other languages. Wrapping it in async lets us use await sleep(millis) wherever needed.

/**
 * @param {number} millis
 * @return {Promise}
 */
var sleep = async function(millis) {
    return new Promise(resolve => setTimeout(resolve, millis));
};
